Welcome to pyGOMoDo's documentation!
====================================
**pyGOMoDo** is a Python library to perform homology modeling and docking specifically designed for human GPCRs. 
**pyGOMoDo** is a python wrap-up of the updated functionalities of GOMoDo web-service (https://gomodo.grs.kfa-juelich.de). 
It was developed having in mind the ist usage through jupyter notebook, where users can create their own protocols of 
modeling and docking of GPCRs.
